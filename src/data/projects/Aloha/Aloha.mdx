## Project Description
**Project Title:** ALOHA: Real-Time Clothing Style Conversion.
<br />
Completed a project focused on developing a real-time clothing style conversion system. The project utilized advanced techniques in segmentation and style transfer to allow users to change the color and style of clothing in real-time. The system features a user-friendly interface that enables users to select from various color and style options, visualize changes instantly, and achieve realistic transformations. The model development process involved stages such as UNet-based segmentation, dataset comparisons between Fashionpedia and K-Fashion, model compression, and final style transfer using advanced algorithms. The project demonstrates the potential for applying AI in the fashion industry, offering practical solutions for virtual try-ons and style experimentation.
