## Project Description
**Project Title:** Edge Computing Based Cloud Parking Management System (Korean License Plate Recognition)
<br />
This project focuses on developing a cloud-based parking management system utilizing edge computing for real-time recognition and processing of Korean car license plates. The system combines YOLOv4 for object detection and LPRNet for high-accuracy license plate recognition, built on TensorFlow. It includes custom functions for detecting electric vehicles and supports TensorFlow Lite (TFlite) for deployment on devices like Jetson Nano, making it ideal for real-time edge applications.
<br />
이 프로젝트는 엣지 컴퓨팅을 활용해 한국 차량 번호판을 실시간으로 인식 및 처리하는 클라우드 기반 주차 관리 시스템 개발에 중점을 둡니다. 이 시스템은 YOLOv4와 LPRNet을 결합하여 높은 정확도의 번호판 인식을 구현하며, TensorFlow 기반으로 구축되었습니다. 전기차 인식을 위한 커스텀 기능을 포함하고, Jetson Nano 같은 장치에서의 배포를 위해 TFlite을 지원하여 실시간 엣지 애플리케이션에 적합합니다.
